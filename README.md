# Nutrients_EDA_Project
ML Assignment 1 
Project: Nutritional Data Exploratory Data Analysis (EDA)

1. Dataset Source
The dataset used in this assignment is the Nutrients Dataset, which provides a detailed breakdown of nutritional values for various food items.

Source: Kaggle / Open ML Repositories.

2. Dataset Overview
This dataset contains nutritional information categorized by food groups (e.g., Dairy, Meat, Fruits, Vegetables). It is used to study the calorie density and nutrient distribution in common foods.

Number of Rows (Original): 335

Number of Rows (After Cleaning & Filtering): 322

Number of Columns: 10


3. Feature Description (Columns)

Food: Name of the food item.

Measure: Serving size (e.g., 1 cup, 100g).

Grams: Weight of the serving in grams.

Calories: Total energy content (kcal).

Protein: Protein content in grams.

Fat: Total fat content in grams.

Sat.Fat: Saturated fat content in grams.

Fiber: Dietary fiber in grams.

Carbs: Total carbohydrates in grams.

Category: The classification of the food group.


4. Why I Chose This Dataset?
Reason: I chose the Nutrients dataset because it contains real world data that is "messy" and requires significant cleaning (like handling the 't' values and unit formats). Additionally, nutrition is a universal topic, making it easier to interpret the results and verify if the correlations (like Fat vs. Calories) make sense in a real-world context.
